there are multiple sibling sgx-table-cell-text tags.  Use the tag containing the attribute "data-column-id" that is equal to "nc".

there are multiple sibling sgx-table-cell-number tags.  Use the tags containing the attribute "data-column-id" that is equal to "lt", "c", "p", "vl", "v", "bv", "b", "s", "sv", "o", "h", "l".
